# 👀 프로그래머스 SQL SELECT 문제 모음

## 1번 - 모든 레코드 조회하기
```MySQL
SELECT *
FROM ANIMAL_INS
ORDER BY ANIMAL_ID;
```

## 2번 - 역순 정렬하기
```MySQL
SELECT NAME, DATETIME
FROM ANIMAL_INS
ORDER BY ANIMAL_ID DESC;
```

## 3번 - 아픈 동물 찾기
```MySQL
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE INTAKE_CONDITION = 'Sick'
ORDER BY ANIMAL_ID;
```

## 4번 - 어린 동물 찾기
```MySQL
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE INTAKE_CONDITION != 'Aged'
ORDER BY ANIMAL_ID;
```

## 5번 - 동물의 아이디와 이름
```MySQL
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
ORDER BY ANIMAL_ID;
```

## 6번 - 여러 기준으로 정렬하기
```MySQL
SELECT ANIMAL_ID, NAME, DATETIME
FROM ANIMAL_INS
ORDER BY NAME ASC, DATETIME DESC;
```

## 7번 - 상위 n개 레코드
```MySQL
SELECT NAME
FROM ANIMAL_INS
ORDER BY DATETIME
LIMIT 1;
```
